# Picker plugin

A Vim plugin, picks lines mathcing pattern into a new buffer

<img src="picker-plugin.png" alt="picker-plugin" width="200"/>

## Install 

Install via Lazy:
```lua
{
    url  = "git@github.com:Yusuzhan/picker-plugin.git",
    cmd="Pick",
    lazy = true,
}
```

## Usage

```
:Pick your_pattern
```
